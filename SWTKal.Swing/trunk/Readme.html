Swing-Benutzungsoberfläche zum Softwaretechnik-Kalender-Projekt

Die in diesem Unterprojekt enthaltenen Klassen wurden im Wesentlichen direkt aus der Calendarium-Anwendung aus dem UML-Lehrbuch von Hitz et al. übernommen (und nur an wenigen Stellen auf die SWTKal-Klassen angepasst)!

So kann auch bei einem frühen Projektstand bereits realisierte Funktionalität interaktiv mit einer entsprechenden Benutzungsoberfläche ausprobiert werden.
